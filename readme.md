# Stinybot

Steps to run:

1. Run perl stinybot.pl
2. ctrl+c to stop

Perl Dependencies:

1. DBI [Installing Perl/DBI support on Unix and Mac OS X](http://dcx.sybase.com/1200/en/dbprogramming/dbd-sqlany-install-unix.html) (install DBI perl module section)
2. Switch
   1. sudo cpan
   2. install Switch
